śrīramaṇa sarasijadaḷākṣa mu
rāri sacarācarabharita duri
tāri nityānaṁda nirjaranikaradātāra
vārijāṁbaka varaguṇāśraya
mārapita vēdāṁtanuta sā
kāra cennigarāya pālisu jagake maṁgaḷava // Naḷa.1.1

śaradhiśayana mukuṁda kaṁsā
suramathana suramaunivaṁdita
giridharācyuta vēṇunāda vinōda bhavadūra
sarasijōdbhava nuta dayāsā

darasa cennigarāya pālisu sakalasajjanara // Naḷa.1.2

varakumāraka vidyevāridhi

kari turaga rathanicaya rājyada

nivahadoḍaneytaṁdanā rōmaśamahāmuniṣa // Naḷa.1.10

bhīra pālisutirda saptadvīpadavanipara // Naḷa.1.25

dhuravijaya dōrbalaru maṇimayamakuṭavardhanaru
caraṇakānatarāgi naḷabhū
varana bhajisutalirdaramarē
śvarana sabheyaṁdadali meredudu nr̥patiyāsthāna // Naḷa.1.26

trāmapuradaṁtesedudalli vidarbhanagaravadu

stōmagaḷa sālinali raṁjisutirdudā nagara // Naḷa.1.27

ballidanu tānāgi nijasuta
rillaveṁdummaḷisutire mana
dalligākṣaṇa baṁdanā damanākyamunivaranu // Naḷa.1.28

baṁda munigavanipati vaṁdisi

ḷeṁda sutasaṁtānavillada siriyadēkeṁdu
noṁdu nuḍiyalu sutara paḍeyada
taṁdegidu saṁtāpavallave
