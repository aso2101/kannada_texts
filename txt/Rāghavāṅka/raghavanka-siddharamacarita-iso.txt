harayōgamaṁ māḍuvaṁ sthūla sūkṣmadoḷu
bharitanāgiha śivananibbaṭṭeyaṁ kaṭṭi
haruṣadiṁ dhyānisuttippātanaikyamaṁ kūḍi śiva tānappanu
nirutaviṁtī bhāṣe husidoḍiṁtī balada
karada naḍuvaṇa beraḷa suḍuvudī guṁḍu suḍa
diralu kaṁḍinnādoḍaṁ naṁbirē kapilasiddhamallēśvarananu // Siddharāma.7.14

sikkadī yōgakke baṟide dhāvatigoḷḷa
dekkaviṁdaṁ hokku hokka mana tamma kā
ryakke sūsade śivana bhāvadiṁ nirṇayaṁ koḍuvudillā dēvana
takka kelasada bhāvadiṁdiralu bhāvavai
kyakke tahudī bhāṣe husiyāge hebbeṭṭu
mikku bēvudu bēyadiralu naṁbire kapilasiddha mallēśvarananu // Siddharāma.7.15

meṟevekkaviṁḍinoḷagaṇa gaṁḍa heṁḍiruga
ḷuṟe tamma hasuḷegaḷiguṁṭe pāl beṇṇegaḷa
neṟevudilladoḍe māṇbudu badukaloṁdoḷḷitaḷidoḍeraḍoḷḷiteṁdu
maṟeyadī dēvana śrīpadava baluviḍidu
toṟeyadavarī dēvanaṁ kūḍuvar pusiye
neṟesuḍugu suṭṭuṁberal suḍadiral naṁbi siddhamallēśvarananu // Siddharāma.7.16

bereyadīyekkaviṁḍinavoḷage strī putra
riralāṟeveṁdaḍavaraṁ biṭṭu haduḷirda
puruṣanuṁ mallināthana kūḍuvaṁ kūḍadire naḍuveraḷ bēyali
puruṣanolladoḍirda satisutaru muktara
pparu husiya darbheya beraḷ suḍali māte pita
rariyāgalirda makkaḷu muktaraharāgadoḍe kiṟuguṇike suḍuvudu // Siddharāma.7.17

hiṁgadī sakaḷa liṁgakke besakeyda ma
rtyaṁgappudaikyapadavāgadirutire balada
muṁgayya taḷa suḍuvudādiliṁgada pratiṣṭheya hōmavaṁ nōḍida
jaṁgamaṁ svargamaṁ paḍevaralladoḍeḍada
muṁgayya taḷa suḍugu suḍadiralu naṁbirē
liṁgamallēśvarananinnu tā guṁḍaneṁdoḍḍidaṁ karataḷavanu // Siddharāma.7.18

halavu tidigaḷalūdi lēsāgi kāsi keṁ
galipa kiḍigaḷanuguḷva paṁcalōhada guṁḍa
nelele kakkasada kammaru niḍidikkuḷugaḷiṁdavaci piḍidu taṁdu
kalikālarudranenisuva siddharāmanaṁ
jaliyaliṭṭaru rāhukētugaḷu balugōpa
doḷu kacci negedarkamaṁḍaḷavanastagirimastakadoḷirisuvaṁte // Siddharāma.7.19

poḍaviyaṁ suḍuva garaḷāgniyaṁ tuḍuki toḍe
veḍeyanaṟasuva mr̥ḍano kaḍalanīṁṭalu kayya
naḍuvuḷida vaḍabanaṁ piḍidagastano śivana niṁdisida dēvēṁdrana
iḍaleṁdu maṁtrāgniviḍida upamanyu sura
roḍeyanaiśvaramaṁ mogeda durvāsanō
nuḍiyalaridenisi divyada guṁḍanirkaiyoḷāṁtanacaḷitasiddhanu // Siddharāma.7.20

tamatamage naḍunaḍugi sākiḷihu samayadā
śrama satya, śuci śūra sahajayeṁdellarā
kramise hō ānitta bhāṣegaḷu saṁduvē eṁdu tannaya śāṁtiya
himadoḷāṟuva guṁḍaniḍikidanu bhāvisade
bhramita tāreyanu nabhaviḍikiteṁteṁbudā
kramavīśanaḍigaruṇa kamala puṣpāṁjaliyanittanō eṁbaṁtire // Siddharāma.7.21

guṁḍu biddoḍe hotti hogedu daḷḷuri masage
kaṁḍu majha bhāpu bhāpapratima nuḍidaṁte
gaṁḍa bhūlōka bhāḷākṣa mahimāśaradhiyeṁdeṁdu maṁdiyaide
koṁḍāḍutire kēḷdu naḍugi tallaṇisi bhaya
goṁḍu dūṣaṇe pathavādalladīśanoḷa
koṁḍippanallennaninnu hogaḷadiriyeṁdā sabhege kaimugidanu // Siddharāma.7.22

ottarisi kādi sōtōḍe kali sūṟevō
dattu taṇiviṁge tanu dayege mana dharmakke
vittu nītige caritraṁ samategaṟivu sāmarthyakke satvaveseva
satyakke nuḍi kr̥pege vadana bhaktige bhāva
vitta naṁbirda māyādi dēhavikāra
sattu keṭṭaḷidavaṁdiṁditta siddharāmaṁ jagake hoṟagādanu // Siddharāma.7.23

vividha satyaṁ māyegatihusika karuṇi pā
pava koluva śāṁtanariṣaḍuvargavene muniva
navaniyoḷu dāni dharmada lōbhi nispr̥haṁ bhaktārtilōkavaṟiye
avinaśvaraṁ parara samayamaṁ geluvano
ppuva jitēṁdriya muktisatisaṁgalaṁpaṭaṁ
śivasiddharāmaneṁdoḷaga ballavaru vaṁdisi tuṣṭarāgutiharu // Siddharāma.7.24

husiyilla huruḍilla haḷivilla hageyilla
hasivilla hajjeyayillaḷupilla muḷisilla
tr̥ṣeyilla madavilla maṟahilla kaṟeyilla kuṁdilla niṁdeyilla
besanavillārtavillāseyilla paḷivi
llusiriḍuvudilla matsaravilla neravilla
vasudheyoḷu siddharāmaṁgeṁba kīrti pasarisitu mūlōkadoḷage // Siddharāma.7.25

pasarisida puṇyavārteya kēḷu muḷidu huru
ḍisi kuṭila vidyāsamudraneṁdeṁba biru
deseva kharparaneṁba jōgi śivasiddharāmana sarvasāmarthyada
pesara dhaṭṭisuve vādisuve sōlisuveneṁ
dosedu hadineṁṭu pīṭhada divya jōgiṇiya
visaraverasaitaṁdanāneviṁḍalisi siṁhada guhege bappaṁdadi // Siddharāma.7.26

hiḍida cakrada gujuṟudaleya śaṁkhada maṇiya
toḍigegaḷa tiriva sōreya hūsidiṭṭageya
huḍiya keṁgaṇṇa niḍiyoḍala bādaṇagiviya paṁcarannada kaṁtheya
kuḍida jommina keṭṭanuḍiya nānājōgi
naḍeverasu kharparaṁ śivasiddarāmayya
neḍege baṁdaṁ baṁdanurivagniyeḍege taṟagele munidu bappaṁdadi // Siddharāma.7.27

aṇiyaradoḷittaleṁdina teladi śivana mē
laṇa bhaktiyiṁ sakaḷa bhūtasaṁkuḷada mē
laṇa karuṇadiṁ muktisatiya mēlaṇa mōhadiṁda maṁtrada mēlaṇa
gaṇisuvādaraṇeyiṁdodavidagnigaḷa mē
laṇa vipuḷa viśvāsadiṁ siddhakula śirō
maṇi māḍutiha nitya hōmamaṁ kaṁḍu keḍisal bagedanā duruḷanu // Siddharāma.7.28

muṭṭi maṁtrākṣateyoḷagnikuṁḍavanu muḷi
diṭṭaḍe nidāghavanamōgha kārgālavada
raṭṭi tāguva teṟadi tāgaluri karagi kiḍiyuḍugi hoge sugidu keṁḍa
keṭṭu maḍuviṭṭukki hoṟasūsuvudakadoḷu
koṭṭāhutigaḷu tēṁkāḍutire ripubaḷa gha
raṭṭa śivasiddharāmaṁ manadoḷaṟidanadhamana kuṭilakāyakavanu // Siddharāma.7.29

jalaja sakhakulajanaṁdidirāda jalarāśi
galagaṁbanugidaṁte koṁḍakke siddhakula
tilakanoyyane lākuḷaṁgoṁḍu puramathananaṁte haṇegaṇṇa beḷaga
holigeyaṁ biḍaludakavaṟatagniyaide gaja
galisi bhugiliṭṭu daḷḷuri baḷḷivaridu hari
delele suṭṭudu suṭṭudā jōgiyaṁ jōgiyoḍanirda parijanavanu // Siddharāma.7.30

iṭṭaṇisi jōgivaḍeyaṁ suṭṭu mīṟi dhā
ḷiṭṭu aṭṭaḷenāḍa kōvūranaidi bhugi
liṭṭu uri habbutiralāvūra kaḍe keḍeda jōgivaḍe modaleḍeyaṇa
daṭṭaisiduriya hoyliṁgāṟadakaṭa bā
yviṭṭu paśumr̥gapakṣi neredoṟalutire karuṇa
huṭṭi kaṇdeṟedu kaṁḍaṁ bēge hoḍedaḍaviyaṁtirda jōgigaḷanu // Siddharāma.7.31

bhūmiya carācarakkolidu māḍuva śāṁti
hōmavēkī prāṇavadheyēke matte ni
rdhūmadhāmākāravāgi niṣkāraṇaṁ kōvūra bēvudēke
sāmarthya hasanāytu bhūtadaye lēsāytu
sōmadharanolavu phalavāyteṁdu śivasiddha
rāmanāthaṁ noṁdu karuṇadiṁ nōḍaleddaṁ jōgivaḍeyoḍeyanu // Siddharāma.7.32

sarasa suṁdara sobaga siddarāmayya su
sthira śuddha śuci śūra siddharāmayya śaṁ
kara sadr̥śa sahajasāmarthyasāgara siddarāmayya rāmanātha
śaraṇasaṁkuḷasuradruma siddharāmayya
śaraṇāgu gurusiddharāmayya rāmayeṁ
deṟagi kaimugidu kārpaṇyamukhanāgirda kharparananīkṣisidanu // Siddharāma.7.33

ēkāraṇaṁ baṁdeyilligelliya kicci
dākasmikaṁ negedu suṭṭudakaṭakaṭa noṁ
dā kuṭila vidyāsamudraneṁbavanu nīnē ninna parijanavanu
ēketteyene beṁda huṇṇa kaṁbiyanikki
nūki kīsadirakhiḷakhūḷanoḷu duruḷanavi
vēkiyoḷu lēsanaṟasade siddhapati karuṇisenageṁdu bāyviṭṭanu // Siddharāma.7.34

karuṇāvalōkanadi siddha haraṇaṁgaḷaṁ
karuvoydu jōgivaḍegabhayavittatti kariku
varida kōvūranūraṁ māḍi kr̥peverasi hāḍi hōmavanu māḍi
eraḍaneya rudranabhinava pitāmahanu su
sthirakara śrīvaraṁ sakaḷa sāmarthyadā
kara siddharāmanāthaṁ śaivasāmrājya sukhadoḷiralēvogaḷvenu // Siddharāma.7.35

inanōḍe navilāḍe virahigaḷu kūḍe bhū
jana hāḍe tr̥ṇamūḍe haṁsegaḷu bāḍe kaḍa
vina sovaḍu tīḍe cādage nōḍe ghanavāḍe dharegaṁbu nīḍe siḍilu
dani māḍe dhare kūḍe beḷegaḷaṁ sūḍe nele
yanu nadigaḷīḍāḍe barpa kārgālavo
yyane moḷetu danidōṟi mīṟi moḷagaṁ hēṟi tōṟittu durdinavanu // Siddharāma.7.36

aḷiya kappanu nīlamēghadasitavanu ka
jjaḷada kaṟeyanu karida katturiya karida kō
kiḷana maḷinava hariya kāḷimeya viṣada karburava nīlada nīlava
eḷedukoṁḍoḍagūḍi tuḷidu hiḷidabujajaṁ
beḷagu muḷugalu baḷidanō bayalaneṁdenisi
daḷavēṟitottarisi mōḍagattale gāḷi hani miṁcu moḷagu sahita // Siddharāma.7.37

oṁdu dinaviruḷoḷā maḷeyoḷu calisadeṁdi
naṁdadiṁ kṣētramaṁ balagoḷuttaṁ liṁga
vr̥ṁdamaṁ nōḍutta saṁtōṣadiṁ baruttiralu siddhara dēvanu
hiṁde nānā jīvajāḷaṁgaḷaṁ koṁda
kuṁdanī taṁdeya padakkeṟagi kaḷeve nā
neṁdu chiṭichiṭalenuttaḍasi kiḍigedaṟutta bedaṟadeṟagittu siḍilu // Siddharāma.7.38

munideṟagiteṁdu bagedellaraṁ koluva nī
cananirisalēkeṁdu piḍidalaginiṁ tiviye
moneyalli siḍilu silukiralu kargattaleyoḷeḍeyāḍutaḍahadaṁte
tanage tāṁ dīvaṭiganādanō giriyōḍi
vanadhiyaṁ hugahugalu tiviye kaitappi vaḍa
bananiṟidu negahidamarēṁdranō eṁdenisi meṟedanacaḷitasiddhanu // Siddharāma.7.39

ugrasarpana phaṇāmaṇiyaṁte kūralagi
nagradoḷu toḍarda siḍilele mahādēva gara
ḷagrīva śiva śaraṇenuttiralu kēḷdu sairisalāṟadahikaṭakanu
vyagradiṁ nijaverasi baṁdidēnai dayasa
magra bhūtasnēhi nīnaḷukadī siḍila
vigrahaṁ nōyaliṟivare maḍakeyuṇakalitaḍāru jīvipareṁdanu // Siddharāma.7.40

baṁdu baṁdellaraṁ koluvudaṁ koṁdaḍē
neṁdiṟidenenalellaraṁ koluva garaḷamaṁ
koṁdenē koraḷoḷirisidenalladenalaṟiyadiṟideneṁdaḍigeṟagalu
saṁda niṣkaṁṭakaṁ nirbhayaṁ niściṁta
neṁdenipa ninage kūralagēvudeṁdu taru
ṇēṁdudharanā siddhapatiyoḍane nuḍidu naḍedaḍagidaṁ liṁgadoḷage // Siddharāma.7.41

beṁda maḷe baralēke siḍileṟagalēkalagi
niṁdiṟiyalēkadīśananuṟahalēke muda
diṁda bhaktara hr̥daya kamaladesaḷgaḷa haseya mēle cittina soḍarina
heṁdaveḷaginoḷu nijataruṇiyaṁ kūḍuvā
naṁdamaṁ toṟedu sukhi sukumāraneṁdenisu
eṁdudharanī maḷeyoḷī caḷiyoḷeytaṁdaneṁdu noṁdanu manadali // Siddharāma.7.42

enage nigrahava siḍiliṁgēṟanīśvarana
manakamarṣanavanittudiṁtideṁdoydu kaḍu
munidu kūralaga bhaṁḍārisi samaṁteṁdinaṁte pradakṣiṇavanu
nenedu barutirpa siddhana padābjada mēle
toneva bālaṁ kusida tale mugida kivi sugida
tanu dainyaviḍida kiladaniverasu baṁdoṁdu kr̥ṣṇaśunakaṁ birdudu // Siddharāma.7.43

karuṇiyanaghane nanna salaheṁdu bidda kā
riruḷa maṟiyō abhayamaṁ bēḍuvajñāna
duruḷiyō maṟedeṟagidaparādhadubbegadi tāṁ noṁdu kaṁdi baṁdu
karuṇisenageṁdu kālviḍiva siḍilō eṁba
pariyaliha śunakaniravaṁ kaṁḍu lōkaika
guru rāmanāthanatibeṟagāgutiha samayadoḷu rātri kaḍegaṁḍudu // Siddharāma.7.44

beḷagu haḷahaḷane moḷetudu timiravaḷavaḷidu
doḷagoḷagaḷalduvu cakōrigaḷu viraha vi
haḷavāṟidavu cakravākavoṟagidavu naidilu kamala kaṇdeṟedavu
aḷiya seṟe biṭṭudu dharājanada kaṇṇa baṟa
tiḷidavu sarōdhi maidegedududayācaḷada
hoḷeva makuṭāgramaṇiyō enisi gaganamaṇi kiraṇamaṁ pasarisidanu // Siddharāma.7.45

pūraipa śaṁkhasaṁkuḷaravaṁ poḍeva bhē
rīravaṁ śuddharāgaṁ gītarava kāha
ḷāravaṁ vividha vādyaṁgaḷa ravaṁ majjanōcitada ghaṁṭāravaṁ
gaurīśa dēvakula cakrēśa cittāva
dhāra jayajayavughē cāṁgu bolleṁba gaṁ
bhīraravaverasu maṁgaḷamayaṁ jīvigaḷabhīṣṭamaṁ koḍutirdudu // Siddharāma.7.46

āga siddhēśvaraṁ tanna padapaṁkajada
mēge horaḷuva sāramēyaniraviṁge beṟa
gāgi nuḍisidoḍe nuḍigoḍalaṟiyadēḷeṁdaḍēḷadoyyane nūṁkalu
hōgadati mūrkhateya māḍuvaḍe kāruṇya

kkēgaiveneṁdenuttiraleṁdinaṁte poḍavaḍalu baṁdaru guḍḍaru // Siddharāma.7.47

poḍamaḍuvaḍaḍiyaneḍegoḍade keḍedoḍaliṁde
maḍidu musukire kaṁḍidēnidēneraḍaneya
mr̥ḍa jagada guruve siddhēṁdra bhūtasnēhi karuṇe līlāvigraha
nuḍi hēḷidārideyalliyadēke baṁdudā

toḍagi hēḷalu toḍagidaṁ trikālajñāni neṟe carācaritabharitanu // Siddharāma.7.48

nereda puṇyada rāśiyō enipa kailāsa
giriya haravariya hiriyaramaneyoḷōraṇaṁ
tara sarisa sāl paṁtivettu kuḷḷirdu hariyajaśakramukhyarāda
surara kinnarara khēcarara garuḍara mayū
rara gaṇēśvarara madhyāsanadoḷaṁbikā
varanadhaṭa madanamadaradani biduvidaḷana mr̥gēśanōlagavittanu // Siddharāma.7.49

ā sabhege rūpakāmaṁ tējabhānu vi
dyāsarasvati kaḷātuhinakiraṇaṁ bhōga
vāsavanenippa gaṁdharvanorvaṁ siriya saṁdaṇiyoḷeṁdinaṁte
dēsevaḍedaitaṁdu rudraṁge maiyikku
vāseyutsavadi tavakadi bharadoḷuravaṇisi
hāsi harahidanu nijatanuva śivanaṁpriya sarōjakeḍabalanaṟiyade // Siddharāma.7.50

vaṁdisi mahāharuṣavaruṣadiṁ nanedēḷa
leṁdāgalanugaiyutire tanna kāl tāgi
hiṁdirdaghōramukhadiṁdogeda ugramukhaneṁdeṁba gaṇanāthanu
noṁdu bhūrmisutelavo garvaviḍidajñāna
diṁdodedeyāgi kr̥ṣṇaścānanāgu hō
geṁdu śāpavanittu pōdanuravaṇisi kōpāṭōpa bharabharitanu // Siddharāma.7.51

mati māsi muda sūsi kaḷeyōḍi kaṇu bāḍi
dhr̥tigeṭṭu maravaṭṭu gatiguṁdi mukha kaṁdi
śitikaṁṭhanātha nuḍi biḍahōgi gaṁṭikkikoṁḍa teṟanādudenage

vasudhe hēsuva jananavādudiṁdeṁdu dū

deseyiṁ viśāpavahudaṁbikāvara karuṇiseṁdu gaṁdharva vīra
besagoṁḍoḍenna janmakṣētra kaivalya
dasu puṇyadikkeyeṁdenipa sonnaligeyoḷa

idu pūrvadōṣavidaṟiṁdāda janmaviṁ

ḍidu kāraṇikavidu śuddhavidu śuciyidu bahiṣṭha vēṣada nijadali
saduhr̥dayadallirdu mallikārjunana dhū
pada mahārōgaṇeya hottinoḷughēyeṁdu

nilladaḍigaḍige dēvara dānavittu maṟa

rellarge besananittaṟiviṁge beṟagāgi hōgi hogaḷuva hiriyara

baṁdu dēvara dānavaṁ koṁḍu gurupadada

ppalli baṁdaṁ liṁgadoḷage neṟe naṭṭadr̥ṣṭiya dhīranenalallama // Siddharāma.9.37

liṁgada mahatva manadiṁ tanuva vēdhisalu
maṁgaḷavenippa beḷagaḷavaṭṭu bhautikake
haṁgigatanavanuḷidu sukhadi meṟeyutta sadyōvarava nīḍuvaṁte
raṁgarītiyineseva vacanadiṁ traijagava
bhaṁgisuta basavarājana jānisuttoṁdu
jaṁgamākāraviḍidiha mahāṁtanu nōḍa baṁda sonnalige purake // Siddharāma.9.38

keṟeya kelasadali siddhēṁdranirutire baṁdu
haṟivillavenuta kareyō oḍḍareṟeyanaṁ
koṟekūligoṁḍanō nāḷe mikkudanageyabāradē eṁbannegaṁ
aṟidanāgaḷe jagajjanada aṁtaryāmi
kuṟuhuviḍiduṁ jaṟevudaṁ kaṁḍu allamaṁ
gaṟuhisuve baḍava maḍagida kaḍavarada teṟada makuṭada maṟeya liṁgava // Siddharāma.9.39

eṁdu nasunagutallamana sammukhake baṁdu
niṁduttamāṁga madhyada vastramaṁ beraḷi
niṁdalōsarise kemmugila saṁdhiyoḷiṁdukōṭaprabhāpaṭalamaṁ
hiṁdugaḷedeseva śaitya prajvalita kāṁti
yiṁdeseva nirupādhika jyōtiliṁgavi
ppaṁdamaṁ bāhyakriyeyoḷu baḷakeyaṁ tōṟalallamaṁ matteṁdanu // Siddharāma.9.40

lōha saṁgadi kiccu baḍihaḍedudeṁbaṁte
dēhasaṁgadoḷātmanāyasaṁbaḍuvaṁte
bēhudoṁdaḷavaṭṭa baḷika biḍalāgadene nereyavarigattu kaṇṇu
hōhaṁda nimagenalu dorakidātaṁ saviya
bēhudē mattārge dorekoḷuvupāyakke
sōheyaṁ koḍalāgadē jagajjanada kr̥pegeṁdanā siddhēṁdranu // Siddharāma.9.41

jaḍara naḍuviralavara saṁgadiṁ kāyaguṇa
viḍidu baḷaluvarenalu nasunaguta gāḷiyaṁ
huḍi hiḍivudē rasane heḍasahude āliyaṁjanavanoḷakoṁḍirpude
poḍaviyoḷu śivayōgi suḷidaḍaṁ nīra naḍu
viḍida kaṁjada teṟadi bhaktiyuktanu alla
mr̥ḍamūrtigagraneṁdenuta hāḍidanu guruvacanavallama meccalu // Siddharāma.9.42

lōkavellaṁ badukaleṁdu nāgārjunana
dēke hōgāḍidaṁ taleyanā vyāḷi baḷi
kēkāda beḷḷi kōrāṁṭakaṁ nīroḷēkaddanenalaṟidudilla
ēkanadvitiyanenisuva liṁgamaṁ naṁba
dā kāraṇaṁ keṭṭaralladavaraḷivarē
sākāraniṣṭhe bhūtaṁgaḷoḷaganukaṁpe tāne parabommavenalu // Siddharāma.9.43

āgalahudadakēnoḍaṁbaṭṭenī yugaṁ
bāgi balagālinuṁguṭavanūṟalkaṁji
yōgigaḷu kadaḷiyaṁ munigaḷanibaru mēru maṁdara guhōdaravanu
bhōgikula sarpalōkakke divijaru divake
sāgarada kaṭṭeyaṁ kaṭṭi rākṣasaru bhaya
rāgirpareṁbāga mānavara naḍuviralu bēḍa siddhēṁdrayenalu // Siddharāma.9.44

adanu nā ballenī yugada jīvara puṇya
dodaviṁge kaikoṁḍa mūrtiyidu husiya mā
ḍidenādoḍuḷida mānavara pariyaisalē allama prabhuveyenalu
cadura ninnī sr̥ṣṭiyē bēṟe adake me
ccidenu nīnirdumilleṁba bayalina bhrameya
bedaṟa bisuḍeṁdu śivasukhagōṣṭhiyiṁda kadaḷiya baṭṭegoṁḍa prabhuvu // Siddharāma.9.45

saṁtaranubhava svānubhavavāgi manadoḷage
ciṁtisida keṟeya madhyada yōgināthana gr̥
hāṁtaradoḷoṁdu gopeyaṁ māḍi naḍuguṁḍinoḷu bāgilaṁ māḍisi
ciṁtisalkaridenipa vajrada kavāṭadiṁ
bhrāṁtugoḷisuve sarva janavanilliheneṁdu
muṁtinaṁtī kṣētradoḷagu hoṟagennadaṁtirutirdu tōrpeneṁda // Siddharāma.9.46

hattaidu divasamaṁ kaḷedu śivakūṭaṇiye
itta bāyenalu hāvina hāḷa kallayya
natyaṁta haruṣamayanāgi puḷakitagātra maiyikki kelasārdaḍe
hattirdavarge besanava koṭṭu kaḷuputta
nityatr̥ptanu tānenippa guruvēkāṁta
cittanāgirdu mattātaṁge manadāyatava hēḷalanuvādanu // Siddharāma.9.47

emma manavoṁdaṁ nivāta gopeyaṁ savesi
summānadiṁda padmāsanava caliyisade
hammamateyaṁ biṭṭu ātmana vicāradiṁ paramakāṣṭheyanneduva
nemmugeya nenenenedu parama śivatatvavanu
bommaveḷagina beḷaguverasi beṟagāgadi
ppummahadabhīṣṭamaṁ ninage hēḷidevenalu dēva binnapaveṁdanu // Siddharāma.9.48

nityatvamuttamāṁgava paḍedudeṁbaṁte
satyasamateya dharege pēḷaleḷasirdaṁte
atyaṁta bhūtadaye avayavaṁbaḍedudō yōgigaḷa cittaveḷagu
pratyakṣa gururūpa dharisi rājisi parama
tatvavenisiha nīnu nenevudēnai jagada
kattaleya kaḷeyaleṁdavatarisi baṁde nī ninage paravēneṁdanu // Siddharāma.9.49

nīnaṟiye mallikārjunanobbanē nitya
jñānamaya sarvajña ṣaḍguṇaiśvaryaṁga
ḷēnuvuṁ koṟateyilladaleseva nenadenna nīnihage kīrtisuvarē
bhānuvina muṁduḷida beḷagesevudē śiva
jñāna ninaguṁṭeṁdu ālōcisidoḍema
gēnuvuṁ tatvanemmugeya hēḷade stautyamaṁ māḍi kaḷedeyenalu // Siddharāma.9.50

dhareyaṟiyutire kapilasiddha mallēśvarana
haravariya nāmavide nā koṭṭenē taṁde
nirupādhika jyōtimaya ninna kāyakke neḷalaḍige hajjeyilla
naralōkadannapānada haṁgu ninagilla
surarasuramānavaroḷinnu munneṁduṁṭe
parabomma nī nimma nenevātanē bommadhāriyenutiṁteṁdanu // Siddharāma.9.51

lōkadoḷu nararaṁ kr̥tārthateya māḍaleṁ
dākāravāda śivasiddhamūruti dēva
sākāra niṣkaḷākāra vēdisalu sarvarige bommada permeya
bēkeṁbanitu sūṟeyēṁ sulabhavāgitte
ēkakkaṭā bayala bhrameyanellarigikka
bēkādaḍikku dēvaru svataṁtraru namō namō eṁdu śuddhanenalu // Siddharāma.9.52

kallayyaneṁda binnahake ummahadi kara
pallavadi maidaḍavi bōḷḷesi ninnaṁte
ballarārū illa manava vaṁcisabahude namagāgadiraveṁbudu

ballidaru adu kāraṇaṁ samādhiya dharege
pallavisi mallikārjunana kṣētradoḷirdu iradaṁte tōrpenenalu // Siddharāma.9.53

adanānoḍaṁbaḍalkārpenē dēva śrī

ṇade neneva mana ninna haṁbaliṁ mattoṁdakeḷasadadaṟiṁdalenage
hr̥dugataṁ nīnu nenavara manave maneyeṁbu

yadane meccuttaṇṇa bommayya kētāra gurugaḷige nēmavitta // Siddharāma.9.54

dhareyaṟiye keṟeya naḍuvaṇa samādhi sthaḷava
paramamaṁgaḷavenalu bēga khananava māḍi
taraviḍidu bhittiyaṁ kaṁbaṁgaḷaṁ mēle kaviva aṟegalla tarisi
hiriyadoṁdemmeguṁḍade adaṟa bāgilaṁ
viraḷavāgiraloḷage tōḍi kīlina kalla
kara celuva kāḷasagaḷaḷavaḍalu māḍi bēgadoḷene bhayaṁgoṁḍaru // Siddharāma.9.55

dēva iddiddidēnēna nenediri illi
yāvuduṁ sthaḷavillavēnu ghātakavāyi
tāva kāryadoḷīga gopege bijayaṁgeydudēke binnapaveṁdene
nīvidakkaṁjalēkānī mahākṣētra
dāvisadoḷu samādhiyoḷacalanāgira

karuvāgi hāḍidī gītamaṁ balleveṁ

hoṁdidī siddharenisuva nālvaraṁ etti paricārakara māḍuveṁ
heṁdada dhareya nr̥para guḍḍaraṁ māḍi keṟe

neṭṭane mahārājikaṁ daivikaṁgaḷaḷa
vaṭṭu lōkadoḷu nānā jīvajāla paḍa
