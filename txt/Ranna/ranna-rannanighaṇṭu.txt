[rannanighaṁṭu]

kuḷir eneśaityaṁ miḷir ene
poḷedudu billendenalke mārambarthaṁ
baḷedun enalk’ eṟagiduduṁ
keḷir ene vājiyaninādam ele kaviratnā // RaNi.1 [kandaṁ]

barey enal atyarthaṁmuṁ
muriy ene kāsida maḷalalkiḷaṅg’ ene geḷasaṁ
murip’ ene maraḷid’ abhāvaṁ
sirivantikey end’ enalke huriyōḍ’ akkuṁ // RaNi.2 [kandaṁ]

hakkiy enal vihagaṁ neṟe
pakkey enal pāsuvuṇḍey end’ ene moḷakā
lkukkene beḷḷakkiyetānē
kekk’ ene diṇḍuviḍidu bappudud’ akkuṁ // RaNi.3 [kandaṁ]

balag’ ene bisuḍ’ emb’ arthaṁ
balamard’ene aṅkadauṣadhaṁ tān akkuṁ
polig’ ene bircugeya teṟaṁ
kelakan enalkikkum allan elekaviratnā // RaNi.4 [kandaṁ]

añcam enal kañjikeyati
bañcaney enemarasiḍakku beḷakene dīpaṁ
poñc’ ene pordidabagevā
neñcal enalk’ añjakal appud ele kaviratnā // RaNi.5 [kandaṁ]

saṇṇad’ enadalkiṟid’ akkuṁ
paṇṇ’ ene māḍ’ emba vāci baḷiy ene vaṁśaṁ
eṇṇ’ ene lekkipuduṁ bisu
gaṇṇ’ ene nalgoravan appan ele kaviratnā // RaNi.6 [kandaṁ]

kōsit’ enal vōṇisidudu ।
sūsidud’ enetuḷḷit’ embuduṁ parimaḷamuṁ
bāsatam ene kampuṁ mige
sāsavam ene sammad’ artham ele kaviratnā // RaNi.7 [kandaṁ]

sugipapum ene sulihaṁ tā
tegaḷ ene pegaḷuḷmit’ end’ enalpudumaṁ tā
bagas’ ene manaviṭṭudumuṁ
negamad’ enal doḍḍit’ appud ele kaviratnā // RaNi.8 [kandaṁ]

aṟil ene tāragetiṭṭ’ en
duṟe balbaiguḷinisu nanasend’ ene diṭamuṁ
gaṟavūv enekēdagepaḍi
yaṟavakkiy eney enalke kōkilan akkuṁ // RaNi.9 [kandaṁ]

kaivaṭṭal saṭṭuge muḍi
poyivekkal poyigilaṁ maḍantaḷejōḷaṁ
kaivale kaṇbalejaṟasuḷi
kaipiḍiy embudane tiḷiye kabbigararasā // RaNi.10 [kandaṁ]

balayaram ene talevāgilu
kalagalam ene kūḍiberasid’ akkuṁ nailaṁ |
galam ene cilvike salesai
gelam ene tāṁ puṇyaśaktiyale kaviratnā // RaNi.11 [kandaṁ]

tōrp’ enekāṇke musurp’ ene
vārp ene kaiviḍidud’ akkum iḷip’ enesōlaṁ
kūrp’ ene niluverag’ ene mō
lārp’ ene susnapam appud’ elē kaviratnā // RaNi.12 [kandaṁ]

ūrp’ ene kīlisiduka
ṇṇārp’ ene nōmpijjugaṁ miḷirp’ ene muripaṁ
kārpoṟey ene maḷegalaṁ
nērpoṟey ene aṟimeyeḷēy elē kaviratnā // RaNi.13 [kandaṁ]

bhaṇḍaṇam enepuyyalvala
baṇḍene jēn perebaḷiyan end’ ene pāṇbaṁ
guṇḍ’ ene neravi tuḷilbaṟi
goṇḍ’ ene nalvīra-rasan elē kaviratnā // RaNi.14 [kandaṁ]

eṟeyaṟeyaṁ toḷa
rpaṟivaṟivaṁsaggigara piriyanuru baḷigaṁ pā
leṟakiluṇivakkideṟaṁ
niṟubaṟegaṁ ponnabasuṟan ele bidi ratnā // RaNi.15 [kandaṁ]

[vr̥॥]
mugilalarēṟidaṁ sirimagaṁ neleveṇṇarasaṁnelakkaraṁ
bagediḷigaṁ mardumbaligan accaram ormodal ōdigaṁ suḷuṁ
begaḍaṟigaṁ sogaṁboṟeganeccaravāḷiga vāḷavāḷigaṁ
tegaluḍegaṁ pogaḷpuḍigan end’ ene nīṁ tiḷi bomman embudaṁ // RaNi.16 [campakamāle]

sompuveṟeyiṁpugaḷevali
gampuḷiligeyuṇisusāsadaḍikiṁ
lgeṁ piḍike bevetuḷirvaḍi
gump’ enetiḷi kaipevaran enippudu ratnā // RaNi.17 [kandaṁ]

aṟiyameyadeya sompusedeya
poṟageḷeyaḷa jēṭadāṇadoṟe kaṇnituva
lguṟipasaḍikōlgebanave
ṇguṟipen āṁ nellimaran enippudu ratnā // RaNi.18 [kandaṁ]

taḷirgaṇeyana balbasadiya
baḷigavaṟige baḷasi naṟake iṅgōlirpiṁ
geḷemugilēḷdoṟey end’ ene
tiḷakabbendeyde beḷḷivesarāṁtavanē // RaNi.19 [kandaṁ]

sāsavadoṭṭilmellaḍi
bānarpeṇevakkivāḍamagalidareḍe kaṁ
ṇbāsaṇikenelanabeṭṭo
ḷbāsuṭitaṇṇuḷil enipud’ elē kaviratnā // RaNi.20 [kandaṁ]

maṟun elesāsavadāṇaṁ
niṟubeḷagene bāyakempu mogasiri pordiṁ
garakeyaralamba neravige
tuṟugippene tiḷige vīḷeyaṁ kaviratnā // RaNi.21 [kandaṁ]

toḷgidan ene poḍavaṭṭaṁ
giḷgidan ene giḍḍanaviḷan end’ ene kuṟubaṁ
puḷgḷidan enekuṟubantaḷa
vaḷgidan ene begaḍugoṇḍavaṁ kaviratnā // RaNi.22 [kandaṁ]

dōṣaviniteḷasadiru nīṁ
lēsāgire pēḷda kannaḍaṅgaḷa tiḷiyal
kāmaramapiḍidoḍ’ āvaṁ
māsaṟasuḷiniṭṭi sadoḍenēsargiḷide // RaNi.23 [kandaṁ]

beḷāyalkuḍu beḷaga
lteḷāṟavubeḷasu modalanōṭam enal miṁ
cuḷābalbasadiya bage
guḷḷāgal naidalaṟige nīṁ kaviratnā // RaNi.24 [kandaṁ]

mugilaralogegaigūṭaṁ
bagedoṟe pāloṟege takkag’ eney oḷd’ āvare
muguḷ’ endaṟi negaḷdavene
mige jājiya beḷḷigūrtavesarāntavanē // RaNi.25 [kandaṁ]

gorb’ ene gaṁbhīraṁ paṟe
parbidud’ ene baḷḷi varidud’ akkuṁ daityaṁ
marbusuḷikārar ene poḷa
durb’ ene siḍil appud’ amala-nijakaviratnā // RaNi.26 [kandaṁ]

jhaḷam end’ enal uṣṇaṁ bā
guḷam end’ eneduḥkhamallaṇikey ene mēḷaṁ ।
paḷag’ end’ ene oṇagidayele
maḷagenebiṭṭiyabaṭṭagasil eneśaṁkhiṁ // RaNi.27 [kandaṁ]

baddaran ene badukidapaṁ
kaddaran enekaḷḷaneddar enal ōḍidapaṁ
biddiganene naṇṭaṁ kai
guddigar ene mallar embud’ ele kaviratnā // RaNi.28 [kandaṁ]

beḷavaravimpu jāgila giḷirpoḍeiṅgaleyūnmadāvaḷi
gayagaldirāḷdavane pūcara balli isājenailaverccu
gaḷogedāromāmpanane golgone vēṭada nīṟam endena
lkaḷeyad kāṇke malligeyenippuda beḷḷivesargetakkanē // RaNi.29 [campakamāle]

callada nele illada muḷi
sollada bageyaṟake bella baḷi ballara sōṁ
kellara gelavallada nuḍi
solladu pūvillaḍir enippudu ratnā // RaNi.30 [kandaṁ]

siriyogedāṇaṁ nailaṁ
bare baṇgoḍi puṭṭu bānvaruṇisiṅgaḍale
ccaṟa bageyaramogasaccara
diriyenepayasābdhi beḷḷivesarāntavanē // RaNi.31 [kandaṁ]

kāyipaṇṇu dāṇamiḷipina
tāyi poḷeniroṭṭinilge bisuḍi bekkaṁ
jāyipu soḍarmaiṅgāṇke
lāyipudejhaḷavāḷke peraḷurvidi ratnā // RaNi.32 [kandaṁ]

navilaragārtiyanaḍedoḷe
saviyuḷimaigoḍuvakki melladibeḷgañca
valuvesu sarena ilpaḷasuva
giḷienalañcebeḷḷivesarāntavanē // RaNi.33 [kandaṁ]

sababāsaneyirpoḍe se
sabeyadapūnailabaggubiḷigginabeḷagiṁ
pubasane bānbaḷiyarakaṁ
ṇgabale tiḷiene dēvabhūruhaṁ kaviratnā // RaNi.34 [kandaṁ]

sogabaḷigisupuvararge
bagasolapige nailasirige mellaḍi bore
ige tagadaḷa seṟavige kaṇbera
loge basaniyenalke lalaneyar kaviratnā // RaNi.35 [kandaṁ]

baḷigelayaṁ badagaṁ nā
ḷgiḷivāḍaṁ nanniḍigaḷanuḷigantaḷegaṁ
saḷiboḷegaḷiyaṁ sājuḷi
geḷeyanenalsagga nuḷidan ele kaviratnā // RaNi.36 [kandaṁ]

nelabalalaṁ niṟi beḷalaṁ
balasolahaṁ baccabaṇise balbaṇivādaṁ
polage susilpeḷagili bā
ḷtolegeleyuṇisārkeyasokey ele kaviratnā // RaNi.37 [kandaṁ]

konarviḍi suḷidāḷkailaṁ
gone nuṇbisugāṇke bagasunile bambalpā
lpane nailasarige pārdeṁ
dene tiḷiniṁpaccad’ eney enippudu ratnā // RaNi.38 [kandaṁ]

mīnāṇmanadaḷeyiṅgai
bānaṁ bopavargaluṇisu kaṇḍerdebāḷ nī
vuṁnāṇkeyakollālpeḷe
tān ene beḷdiṅgaḷ embud’ ele kaviratnā // RaNi.39 [kandaṁ]

ālenabalsaravalagala
vāliṅgaḷelendu pakkavane bilneraṁ
kōlāṅke tān enal maṟu
gōl ene billāriy embudaṁtiḷi ratnā // RaNi.40 [kandaṁ]

sūḍaguresekkebeṅgone
gūḍaḷaṟal suḷibedarcu pēṟebeṅgi
ḷḷoḍemuṭṭaṟimekaḷalaḍi
vāḍenebiṅgey embudaṁtiḷiratnā // RaNi.41 [kandaṁ]

miḷirele pōparāṇmanaḻevaṭṭigariccheganaṇpuvāḍigaṁ
poḷaṟikeyilladaṁ savalan eccaṟameyaṟanailavaḷkegaṁ
jhaḷaviligaṁ miḷirvalalanendenalindrana rañcu tuḷtugaṁ
piḷivaḷigōlan end’ ene khalaṁ bage sāsava cannavāḷavāḷ
kuḷiralevaṭṭidōrke magaḷ end’ ene munnaṟigauriyembudaṁ // RaNi.42 [campakamāle]

peṟage baḷakkayesurpaḍe
yaṟadaḷemaguḷaḷate kāṇbeiṁnaḍe
neṟedāḷaḷaraḷikepoḷtuḷadikeyeṁ
daṟigoḷmalt’ endu beḷḷivesarāntavanē // RaNi.43 [kandaṁ]

pottikk’ ene poredāḷdiha
ruttikk’ ene kūḍebāḷdaparpoṟepāḷaṁ
pattikk’ ene sārdapareṁ
buttiya nīṁ tiḷiyobeḷḷivesarāntavanē // RaNi.44 [kandaṁ]

suḷivaḷisāre gālsarananāḷkeya kaṇbasenailidōṟupe
ccaḷageṟey end’ enaldyunadikēḷkēnirdumbaḷasoldebaṁṇaṇaṁ
kuḷirede kailanaṇpagare tiṅgaḷavaṭṭara // RaNi.45 [campakamāle]
