[kaṁ॥]
śrīgē jayaśrīgē vadhūḥ
śrīgē manōramānēnisi sakaladharitrī
bhāgade negaḻd’ irdan di
gbhāgadoḷaseyalke kīrti sāhasabhīman // SāBhīVi.10.1 [kandaṁ]

[va॥]
āgaḷ murāntakan samaravijayiyappa sāhasabhīmana sāhasakke
mecci dharmanandanana mogaman nōḍi

[caṁ॥]
kurupatipātadin kr̥tabhayaṅkaramarjunabāṇaghātaja
rjaritaśarīrasyandhavamadāttabalāmbudhiyuṁ niśātatō
marahatakarṇamastamitaśalyaśarasthitabhīṣmasāhasan
doreyenisittu bhāratakke gelda raṇam kumārāṅkarāmanā // SāBhīVi.10.2 [campakamāle]

[caṁ॥]
turagakabandhadoḷ naraśiraṅgaḷa vīramahākabandhadoḷ
kariśiramāgalutkaṭakabandhadolā bhaṭaran taguḷpoḍā
turagamukhambolādudu vināyakanantevolāytu nōḍa
kinnararaṇamaṇḍōdēvogaldudā raṇaman kumārāṅkarāmanā // SāBhīVi.10.3 [campakamāle]

[caṁ॥]
samadavirōdhihastadoḷe tadrudhirāmbuvinimajjanārdraka
darmaviṣughātapātitaśirōvanipālanaṭatkabandhavi
bhramamatiraudramāytu sumanassumanōbharanityalampaṭa
bhramaramanēkabhūtaniḷayam samaram kumārāṅkarāmanā // SāBhīVi.10.4 [campakamāle]

[caṁ॥]
kurudhareyan nijōgragadeyērgaḷinuttu samastavairibhū
para rudhirāmbuvinda bedemāḍiya ghaṭṭakumbhimuktabhā

[p. 153]
suranavamauktikaprasarabījamanōḷiye sūsi tārabhū
dharasitakīrtiyemba beḷesan beḷedan parasainyabhairavan // SāBhīVi.10.5 [campakamāle]

[caṁ॥]
nuḍidudajoḷ toḍalnuḍidarendoḍamīyade poṇṇa pūṇke mā
ṇpoḍamijivalli meygegidarendoḍe munname nānbuvan mogaṁ
guḍanavagendoḍēn nuḍiya bāgada bīrada kīrtivalli dāṁ
guḍiviḍe pēḻavēḻpudo guṇatrayaman kumārāṅkarāmanā // SāBhīVi.10.6 [campakamāle]

[caṁ॥]
vasudhege mēreyādudu payōdhi payōdhige mēreyāduda
ṣṭese desegaḷge sīme digibhāvaḷiyāytane māṇadelliyuṁ
pasarisi tāne vāridhiyanatta diśāḷiyanatta diggaja
prasaradinatta nīldudu yaśaḥprakaram kumārāṅkarāmanā // SāBhīVi.10.7 [campakamāle]

[kaṁ॥]
iḷeyoḷadaṅkurisi hari
ttaḷadoḷ vardhisiye nīldu gaganakkasuhr̥
tkuḷanārī nayanāmbuve
beḷeyisidudu kīrtilatikeyan sattiganā // SāBhīVi.10.8 [kandaṁ]

[va॥]
endu murāntakan pogaḻdu kurukulāntakana mogaman nōḍi

[kaṁ॥]
muniyisidarātiyan ko
lvenendu pūṇvante pūṇkedappade kolaluṁ
ninagallade tīrgume ge
lvenendu nuḍidante geluvamijivabeḍaṅgā // SāBhīVi.10.9 [kandaṁ]

[kaṁ॥]
peṟarāḷa peṟara kudureya
peṟarāneya peṟara rathada pērvaladindaṁ
ijivar peṟarēkāṅgadi
nijivavarār ninna teṟadinijivabeḍaṅgā // SāBhīVi.10.10 [kandaṁ]

[kaṁ॥]
javanuṇi barcidavol kau
ravarallara kondu kauraveśvarabalaman
taveyiḍidu tīrcalēkāṁ
gavīra ninnannanāvanijivabeḍaṅgā // SāBhīVi.10.11 [kandaṁ]

[ma॥sra॥]
mane peṇbaṁ makkaḷembī nuḍi masi nijakr̥ṣṇāṅgadoḷ puṭṭi kīrtyaṁ
gane śubhracchāyeyādaḷ baḷedaḷanilamārgambaraṁ parbidaḷ di
gvanitāsandōhamaṁ tāpinamaḍiyiḍuvaḷ gōtraśailāgradoḷ ma
jjanamā vōmāpagāvāriyoḷene piriduṁ bhumbhukam sāhasāṅkan // SāBhīVi.10.12 [mahāsragdhare]

[va॥]
endu nārāyaṇan cāḷukyanārāyaṇana śūrakr̥tyaṅgaḷan
pogaltuttirpinamāgaḷ dhr̥tarāṣṭranuṁ gāndhāriyuṁ baral
dharmanandanan poḍemaṭṭu

[u॥]
īyada bāḍamayyanemagollade sandhiyanayya nimma gāṁ
gēyara mātugēḷadetidan kururājanadarke duḥkhadin
bāyuṭiyalkevēḍa nimagān maganallene nīma tandeyuṁ
tāyumamōghaminnemage pāṇḍumahīpatiyindamaggaḷam // SāBhīVi.10.13 [utpalamālike]

[va॥]
embudumā prastāvadoḷ duryōdhanana mādēviyarappa
bhānumatiyuṁ candramatiyuṁ modalāda atirathamahārathārdharatha
pramukhasāmantamaṇḍalikāntaḥpurakānteyarondāgi tantamma jīvitē
śvararan saṅgrāmaraṅgadoḷaṟasuttuṁ bandu kāṇade

[ma॥]
gaḷitāśruprakaram payōdharataṭavyastāmbaram srastakuṁ
taḷabhāram parigeṭṭa dīnavadanam nirghātaniśśvāsamā
kuḷacittam parivēṣṭitapracurahāhākrandanam śōkavi
hvaḷadin kolludigoṇḍudā kurukulakṣmāpālanantaḥpuram // SāBhīVi.10.14 [mattēbhavikrīḍitaṁ]

[caṁ॥]
śarakoḷe satta yōdhapatigaḷ paḍe vāraṇapātadinda cā
tura caturaṅgasēne keḍe bhūruhavillada baḷḷigaḷtaden
doreyenisitto bhīmabhujanirdaḷitātmanarēndrarilladu
bbarisida śōkadindaṟali kōṭelegoḷta tadaṅganājanam // SāBhīVi.10.15 [campakamāle]

[va॥]
antu śōkavihvalarāgi barpalli kauravanitambiniyar gāndhārī
dhr̥tarāṣṭrara kāla mēle kavidubiṭṭu


[p. 155]
[u॥]
ellidanellidan kurumahīpatiyillidanellidan mahī
vallabhaninduvaṁśatilakāhvayanellidanellidan lasa
tpallavakīrti cārutaramūrti phaṇīndrapatākanalli tā
nellidanellidan gaḍa manaḥpriyanallidanō suyōdhanan // SāBhīVi.10.16 [utpalamālike]

[kaṁ॥]
bhānumativadanabisaruha
bhānu lasaccandramatimukhēndīvaracaṁ
drānananallidanō ravi
sūnumanaḥpriyanenippa kurukulatilakan // SāBhīVi.10.17 [kandaṁ]

[va॥]
endu karuṇākrandanaṅgeyyudumāgaḷadan kaṇḍu dharmanandana
nandharājaṅgaṁ gāndhārigaṁ poḍemaṭṭu—

[va॥]
nimage besakeyvamayyar
śramadammin kūḍikoṇḍu nelanan pūrva
kramadindāḷvim nō
ḍi maṟappudin nimma sutaranandhanr̥pālā // SāBhīVi.10.18 [kandaṁ]

[va॥]
endu dharmanandanan nuḍiya dhr̥tarāṣṭranavadharisi—

[kaṁ॥]
emaginnuṁ mānasavā
ā mōhamillindu ninnaninitaṁ bēṭpim
yamanandana asuvosadar
sumakkaḷan kuḍuvudevara vaniteyargīgaḷ // SāBhīVi.10.19 [kandaṁ]

[va॥]
ene id’ āva gahanamavagaḷgaṁ saṁskāravidhiyin saṁskāraman
māḻpen end’ āgaḷ

[kaṁ॥]
tanuśōkānaḷanaḷural
dinēśasutanaṅgamahitaḷādhipa rādhā
tanayana kaḷebaraman pōṁ
kane kaṇḍaḷ kunti tanna coccila maganan // SāBhīVi.10.20 [kandaṁ]

[va॥]
antu karṇana kaḷebaraman taḷkaisi putranan kaṇḍa teṟadin
paṟayisuva nijajananiyan kaṇḍu pāṇḍunandananidēnendu
besagoḷḷe

[ma॥]
vanajātapriyabandhugaṅgapatiyan dharmaṅge ninnan hutā
śanamitraṅge bakāriyan kulaśigaṁ gāṇḍīviyan pettenā
dinakr̥nnandananenna coccila magan nimmagrajan karṇanā
ninitan nimmaya mōhadindusiradīrden kondenay dharmajā // SāBhīVi.10.21 [mattēbhavikrīḍitaṁ]

[va॥]
embuduṁ dharmanandanādigaḷatyantaduḥkhitarāgi intendar

[ma॥]
piriyaṇṇaṁ kramadātanarkasutanendēkinnagaṁ pēḷidi
dirō mun pēḻdoḍe mātegā piriyanemmaṇṇaṅge sāmrājyabhū
bharaman koṭṭavanāḷvoḍande baṭikāvāḷāgi saubhāgyadiṁ
direvē molladugeydarendu jaḍidar kuntīsutar kuntiyaṁ // SāBhīVi.10.22 [mattēbhavikrīḍitaṁ]

[va॥]
embuduṁ nārāyaṇan kṣatriyadharma[marmadindādu]dan sairisuvu
dendavaraṁ vigataśōkar māḍe [samāhitacittarāgi] duryōdhanaprabhr̥tigaḷa
peṇaṅgaḷanondeḍeyoḷiṭṭu dvijarin saṁskārakarma[man nirvartisi]
karṇana kaḷebaraman brāhmaṇarin tāve saṁskāraman māḍi samastagaṁ
jaladānakriyegaḷan nirvartise tadanantaraṁ kontiyuṁ gāndhāriyuṁ
dhr̥tarāṣṭranuṁ kurunarēndranarasiyar modalāgellarumanaśanadīkṣeyan
kaikoṇḍuttamamōkṣaman tāldidar.

ittal dharmanandanan mukundaṅge kayyan mugidu. 

[ma॥]
initondugrasamastabhāratamahābhāratiyuddham mukuṁ
da nijānugrahadindamāytu kiṟiyan pārthan mahīmōhami
llenagītan kuruyuddhalabdhajayanī bhīmaṅge nīṁ paṭṭabaṁ
dhanaman māḍbudedarke hastinapuraprasthānaman māḍuvam // SāBhīVi.10.23 [mattēbhavikrīḍitaṁ]

[va॥]
embuduṁ bhīmaṅge paṭṭabandhaman māḍbudendu vāsudēvan muntāgi
sahadēvan nirūpisida śubhamuhūrtadoḷ hastinapuraman pokku

[ma॥]
deseyan tekkane tīve tāṭatamahābhērīravam viprare
ṇṇeseyoḷ vēdaninādadin parase paṅkējākṣiyar kūḍi saṁ
tasadin sēseyanikke pāṇḍavabalaprākāravīraṅge sā
hasabhīmaṅge manōmudam berasu kr̥ṣṇan kaṭṭidan paṭṭaman // SāBhīVi.10.24 [mattēbhavikrīḍitaṁ]
